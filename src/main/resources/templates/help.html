Aide Service REST

URL Méthode Opération Retour
/ GET Affiche la page d'accueil Format HTML
/help GET Affiche la page contenant les informations d'aide Format HTML
/cv24/resume/xml GET Affiche la liste des CV stockés en XML Flux XML
/cv24/resume GET Affiche la liste des CV stockés en HTML Page HTML
/cv24/xml?id=<id> GET Affiche le contenu complet du CV dont l'identifiant est transmis en paramètre Flux XML conforme au schéma xsd cv24
/cv24/html?id=<id> GET Affiche le contenu complet du CV dont l'identifiant est <id> Page HTML
cv24/insert POST Ajout d'un CV dans la base Flux XML retour avec statut
/cv24/delete DELETE Suppression du CV dont l'identifiant est <id> Flux XML retour avec statut
/cv24/search?... GET Recherche la liste des spécifications répondant aux contraintes exprimées dans la requête Flux XML comprenant la liste des articles répondant aux critères de recherche exprimés
